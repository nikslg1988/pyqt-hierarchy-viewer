# PyQt Hierarchy Viewer

Тестовое задание на позицию **Junior Python Developer**.

Приложение отображает иерархическую структуру из SQLite базы данных с использованием **PyQt5** и **QTreeView**.

---

## Функциональность

- Загрузка данных из SQLite
- Построение иерархического дерева
- Отображение элементов через `QTreeView`
- Цвет фона элемента в зависимости от поля `state`
  - `0` — красный
  - `1` — жёлтый
  - `2` — зелёный
- Отображение изображения из BLOB (`PNG`)
- Редактирование имени узла
- Добавление дочерних элементов через контекстное меню

---

## Технологии

- Python 3.10
- PyQt5
- SQLite
